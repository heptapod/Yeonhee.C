Featured Projects

AR Pet Interaction System

An augmented reality application that uses dual marker tracking to create interactive experiences with virtual pets. The system recognizes bottle and hand markers to enable petting interactions, demonstrating advanced AR tracking capabilities and user experience design.

Technical Features:

  • Dual marker tracking system using AR Foundation
  • Proximity-based interaction detection
  • State management for pet behaviors (sleeping/awake states)
  • Cooldown system for natural interaction flow
  • Real-time sprite switching based on interaction
  • 3D positioning and scaling relative to marker size

Technologies Used:

Unity, AR Foundation, C#, ARCore/ARKit, Computer Vision
